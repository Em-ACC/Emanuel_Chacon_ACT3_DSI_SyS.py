# Actividad Formativa 3 - Filtros Digitales

Proyecto desarrollado en Python para la materia de procesamiento de senales digitales.  
El objetivo es disenar y evaluar filtros digitales pasa bajos, pasa altos y pasa bandas utilizando librerias de Python como SciPy, NumPy y Matplotlib.

---

## Objetivos del proyecto

- Generar senales de prueba con diferentes frecuencias y ruido
- Disenar filtros digitales FIR e IIR
- Aplicar los filtros y comparar los resultados
- Analizar la respuesta en frecuencia de cada filtro
- Evaluar la efectividad de los filtros en la reduccion de ruido

---

## Herramientas utilizadas

- Python 3.x  
- NumPy  
- SciPy  
- Matplotlib
